# Crystal Resort Internal Tools

This repository contains internal tools for Crystal Resort operations.

## Structure

```
crystalresort-app/
├── dashboard/     # Main dashboard with links to all tools
├── roomstatus/    # Room Status management tool
├── purchase/      # Purchase Board tool
└── shared/        # Shared configurations (Supabase/Firebase)
```

## Tools

### Room Status
Access the Room Status tool at `/roomstatus/`

### Purchase Board
Access the Purchase Board tool at `/purchase/`

### Dashboard
Access the main dashboard at `/dashboard/` to navigate between tools.

## Getting Started

Each tool is self-contained. Refer to the individual tool's documentation for setup and usage instructions.
